用易语言制作的桌面弹幕程序？！

2025/8/19

介绍

这是使用易语言制作的一个能把“弹幕”发送到 Windows 桌面的程序。
介绍视频（哔哩哔哩）：

下载

可从 GitHub 下载程序 (exe) 及其源码 (e)：
https://github.com/wsrj/edeskdanmaku

原理

视频里说好的下期揭晓，拖了一年多了还没做……
实际上就是在一个单独的窗口里放置一个透明标签当作“弹幕容器”，再根据输入的文本设置标签的标题和颜色，使用 Windows 编程接口抠除窗口的底色达到“弹幕”效果，然后使其在屏幕上滚动。
你们可以下载源码自己研究学习。请勿用于不正当用途。
